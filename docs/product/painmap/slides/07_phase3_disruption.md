# Slide 07 — Phase 3 破壞式機會辨識（Gagné Event 4：找到位置上）

---

## Action Title

**Phase 3：你不需要比他們強。你需要找到他們不屑做的事。**

## 內容

讓我說一個反直覺的事。

大公司不是打不贏你。

是**打贏你對他們沒好處**。

（停頓）

你的市場太小——他們的業務團隊不屑。
你的價格太低——他們跟進會拉低利潤。
你砍掉的功能——他們的企業客戶不允許砍。

**這就是你的機會。**

---

兩種模式。只有兩種。

### 低端破壞

「你的工具太貴太複雜。我只需要其中 10%。」

目標：**被過度服務的人。** 付太多錢，用不到那些功能。

### 新市場破壞

「我想解決，但根本沒工具。」

目標：**非消費者。** 想解決但買不起，或根本不知道有工具。

---

怎麼判斷你的機會是哪種？

**看替代方案。**

你的目標客群，現在怎麼解決這個問題？

| 他們現在的解法 | 訊號 |
| :--- | :--- |
| 花大錢用複雜工具 | 低端破壞 |
| 用 Excel + 手動 + 忍耐 | 低端破壞 |
| 完全沒有解法 | 新市場破壞 |
| 免費工具但痛苦地用 | 新市場破壞 |

## 視覺設計

- 第一幕：「打贏你對他們沒好處」——白字黑底，巨大，居中
- 第二幕：三個原因逐行浮現
- 第三幕：雙欄對比（低端 vs 新市場），用兩種對比色
- 第四幕：替代方案判斷表
- 整體感覺：揭秘。像在告訴你一個秘密。

## 講師備註

> 這一頁的核心是那句話：「打贏你對他們沒好處。」
> 說完停 3 秒。讓它沉澱。
> 「這是哈佛教授 Clayton Christensen 花了 20 年研究的結論。但用白話說就是這一句。」
> 兩種模式不用細講理論。各用一句話 + 一個問題就夠。
> 「低端：誰在用太貴太複雜的工具？新市場：誰想解決但根本沒東西用？」
> 「選不了？選低端。因為低端有明確的競品可以對比，你接下來畫策略圖會更容易。」
