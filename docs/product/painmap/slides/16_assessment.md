# Slide 16 — 自我評估（Gagné Event 8）

---

## Action Title

**出門前。三個問題。**

## 內容

不是考試。是你問自己。

---

### 問題一：你記住了什麼？

```
□  7 步：看清 → 找位置 → 出手
□  物理量：具體的人、付錢的痛、手工交付、收款管道
□  低端破壞 vs 新市場破壞
```

記住一半就夠。Prompt 資料夾會幫你記剩下的。

---

### 問題二：你拿到工具了嗎？

```
□  知道 Prompt 在哪裡
□  知道 {{}} 是自己填的
□  知道每步有什麼 AI 做不了
```

---

### 問題三：你能出手嗎？

```
□  你有至少 1 個痛點的一句話描述
□  你能說出 1 個認識的人的名字
□  你知道回家第一步做什麼
```

---

如果第三組一個都打不了——

等一下下課找我。我們聊 5 分鐘。

---

如果你只記住一件事，記這個：

> **找一個你認識的人。問他：「如果我幫你做 X，你願意付 Y 嗎？」**

整套框架。濃縮成這一句。

## 視覺設計

- 三組問題逐組 build
- 每組用不同顏色背景（藍/綠/橙）
- 最後的濃縮引用：全螢幕，最大字體
- 沒有裝飾。文字即力量。

## 講師備註

> 「不用全部打勾。但最後一組——如果一個都打不了，找我。不是你的問題。可能只是需要一點引導。」
> 最後的濃縮句，慢慢唸。每個字停頓。
> 「找一個你認識的人。問他。如果我幫你做 X。你願意付 Y 嗎。」
> 「就這樣。就這麼簡單。」

## 過場

→ 「最後一件事。」
