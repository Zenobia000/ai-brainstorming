# Slide 11 — Phase 7 GTM 一頁紙（Gagné Event 4：出手驗證下）

---

## Action Title

**Phase 7：一頁紙。72 小時。第一塊錢。**

## 內容

六個 Phase 的所有產出，收成一頁 A4。

就一頁。多了你也不會看。

```
┌─────────────────────────────────────┐
│  問題（一句話）                       │
│  誰（具體到能打電話）                 │
│  只做 / 不做 / 大公司不跟進因為       │
│  賣什麼 / 怎麼做 / 多少錢             │
│  前 10 個客戶（3 個有名字）           │
│  72 小時：每天 3 個動作               │
└─────────────────────────────────────┘
```

---

72 小時行動計畫。

不是「計畫」。是「明天起床第一步做什麼」。

| | 動作 | 目標 |
| :--- | :--- | :--- |
| **Day 1** | 發 LINE 給小美 | 約 30 分鐘通話 |
| | 發 Messenger 給阿珍 | 拿到回覆 |
| | 開好收款連結 | 連結可用 |
| **Day 2** | 跟小美通話 | 列出 10 個常見問答 |
| | 手工做一次 | 幫小美設定好 |
| | 在 FB 社團發文 | 3 個回覆 |
| **Day 3** | 確認效果 | 拿到回饋 |
| | 發預售訊息 | **≥ 1 人付款** |
| | 收錢 | 或記錄拒絕原因 |

---

注意看。

每一個動作都是**物理動作**。

發訊息。打電話。做模板。收錢。

沒有「研究市場」。沒有「思考策略」。沒有「再看看」。

---

72 小時後，你會知道兩件事之一：

**有人付錢了。** → 你找到了種子。

**沒人付錢。** → 你知道為什麼。這比「不知道做什麼」好一百倍。

## 視覺設計

- 第一幕：一頁紙結構（信封視覺，極簡）
- 第二幕：72 小時 timeline（三天，水平，Day 3 最大）
- Day 3 的「≥ 1 人付款」用綠色高亮 + 放大
- 最後兩個結果：雙欄，左綠右橙，都是正面

## 講師備註

> 「注意看 Day 1 第一步——發 LINE 給小美。不是做 logo。不是選技術。是發訊息。」
> 「最容易忘的：收款連結。如果有人回『好』，你 5 分鐘內能發付款連結嗎？如果不能——Day 1 就要設好。」
> 「72 小時後不管結果如何，你都贏了。有人付錢是贏。沒人付錢但知道原因——也是贏。唯一輸的是什麼都沒做。」

## 過場

→ 「七步講完了。但知道不等於做到。讓我教你怎麼用。」
