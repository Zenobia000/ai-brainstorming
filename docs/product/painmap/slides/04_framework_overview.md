# Slide 04 — 框架全貌（Gagné Event 4：呈現內容 — 總覽）

---

## Action Title

**7 步。從模糊到收錢。**

## 內容

一條線。七個點。

```
  看到 → 看透 → 找角落 → 畫策略 → 最小品 → 確認 → 出手
  P1     P2     P3       P4       P5      P6     P7
```

三個階段。

| | 你在回答什麼問題 | 花多久 |
| :--- | :--- | :--- |
| **看清問題**（P1-2） | 「到底什麼問題？」 | 1-2 小時 |
| **找到位置**（P3-4） | 「我站哪裡？」 | 1-1.5 小時 |
| **出手驗證**（P5-7） | 「最小能賣什麼？」 | 1-1.5 小時 |

全程 3-5 小時。不是 3-5 個月。

---

（停頓）

但有四條紀律，貫穿始終。違反任何一條，流程就會失效。

1. **沒有分數。** 不打分、不排名、不預測成功率。
2. **說「問題」，不說「點子」。** 「我的點子」讓你防衛。「這個問題」讓你客觀。
3. **AI 做苦力，你做判斷。** 永遠不要問 AI「我該不該做？」
4. **每一步都要一個名字。** 填不出真人名字？先去認識人。

## 講師備註

> 指著流程線：「七步。記不住沒關係。記住三個階段就好：看清、找位置、出手。」
> 「3-5 小時」要特意強調：「不是三個月。是一個下午。」
> 四條紀律逐條講，每條一句話解釋。
> 第四條最重要：「如果你在任何一步填不出一個人名——你不是方法有問題。你需要先去認識人。這不是壞事。這是你現在最該做的事。」
> 停留讓學員拍照。
