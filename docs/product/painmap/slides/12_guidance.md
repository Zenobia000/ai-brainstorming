# Slide 12 — 學習引導（Gagné Event 5）

---

## Action Title

**用 AI 跑這套框架，三件事你必須知道。**

## 內容

### 第一件：`{{}}` 是你填的。不是 AI。

Prompt 裡的雙大括號——那是你的真實資料。

你的人名。你的觀察。你的數字。

如果你讓 AI 連這些都幫你編——你不是在用工具。你是在自我欺騙。

---

### 第二件：每個 Phase 都有一條 AI 過不去的線。

```
Phase 1 → 「我認識這個人嗎？」     ← 你答
Phase 2 → 物理量畫布               ← 你填
Phase 3 → 選哪種破壞模式           ← 你選
Phase 4 → 不做清單確認             ← 你定
Phase 5 → 桌面演練                 ← 你跑
Phase 6 → 最終判斷                 ← 你下
Phase 7 → 按發送                   ← 你按
```

AI 做苦力。你做判斷。這條線不能模糊。

---

### 第三件：紅隊 Prompt 是用來打你自己臉的。

Phase 4、5、7 都有紅隊 Prompt。

用法很簡單：

1. 跑完主 Prompt。拿到產出。
2. 把產出貼進紅隊 Prompt。
3. AI 會找你的漏洞。
4. **不要辯解。調整。**

如果紅隊讓你覺得整個策略都有問題——

**恭喜。你在花錢之前發現了。**

## 視覺設計

- 三件事逐一 build
- 第一件：`{{}}` 符號放大，紅色
- 第二件：七行清單，「你」字用黃色高亮
- 第三件：盾牌 icon，1-2-3-4 步驟
- 「恭喜」用反轉色（通常壞消息的顏色寫好消息）

## 講師備註

> 語速快一點。這是工具使用說明，不需要太多情感。
> 「最重要的一句：AI 的輸出是草稿。你的判斷是答案。」
> 「紅隊那部分——很多人跑了一次主 Prompt 就停了。不要。多跑一次紅隊。它會打你臉。但臉腫了比錢沒了好。」
