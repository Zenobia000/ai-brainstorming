# Slide 01 — 引起注意（Gagné Event 1）

---

## Action Title

**你做了三個月，卻從來沒問過任何人：「你願意付錢嗎？」**

## 內容

畫面只有一個數字：

# 90%

然後一句話浮現：

> 90% 的創業者，做了 logo、選了技術、寫了 landing page——
> 卻從來沒有發出一封訊息給真人。

（停頓）

這叫什麼？

我們給它一個名字。

### 代理動作。

| 看起來像進展 | 實際上離第一塊錢更遠 |
| :--- | :--- |
| 做 logo | 沒人知道你存在 |
| 選技術棧 | 沒有驗證問題存在 |
| 寫商業計畫書 | 沒人看過你的東西 |
| 研究市場規模 | 沒跟任何真人說過話 |

這些全部是代理動作。

它們讓你覺得「我在前進」。但你沒有。

## 視覺設計

- 第一幕：純黑背景 + 白色巨大 "90%"（佔螢幕 60%）
- 第二幕：90% 縮小，下方浮現一句話
- 第三幕：「代理動作」四個字，居中，紅色
- 第四幕：對比表格滑入
- 動畫節奏：慢。每一幕之間停 2 秒。

## 講師備註

> 開場就丟數字。不寒暄、不自我介紹。
> 「90%」亮出來的時候，什麼都不說，等 3 秒。
> 然後問：「你們猜這個數字是什麼？」
> 不等回答：「90% 的人跑完創業流程，卻從來沒發出一封訊息。」
> 掃一眼台下：「在座的，有沒有人做過這些事？做 logo？選技術棧？」
> 舉手的人會很多。
> 「沒關係。今天之後不一樣。」
> 「今天結束後 72 小時，你會發出你的第一封預售訊息。不是計畫。不是研究。是一封發給真人的訊息。」

## 過場

→ 「怎麼做到？我告訴你。」
