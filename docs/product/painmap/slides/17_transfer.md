# Slide 17 — 強化記憶與遷移（Gagné Event 9）

---

## Action Title

**72 小時。從現在開始算。**

## 內容

不是下週一。不是「有空的時候」。

**從你走出這個門開始。72 小時。**

---

### Day 1：看清問題。

跑 Phase 1-3。拿到破壞機會卡片。

### Day 2：找到位置。

跑 Phase 4-6。拿到最小產品設計。通過安全檢查。

### Day 3：出手。

跑 Phase 7。發出你的第一封預售訊息。

---

你需要的全部工具，都在這裡：

| 東西 | 用途 |
| :--- | :--- |
| 7 份 Prompt | 複製貼上，跑完整個流程 |
| 使用指南 | 卡住的時候看 |
| 產出範本 | 不知道產出該長什麼樣的時候看 |

---

72 小時後你會知道兩件事之一。

有人付錢了——你找到了一個生意的種子。

沒人付錢——你知道了為什麼沒人付。

**兩個都是贏。**

唯一輸的是：什麼都沒做。

---

所以。

現在在心裡決定一個日期。

> 我要在 ___月___日 之前，發出我的第一封預售訊息。

記住這個日期。

## 視覺設計

- 「72 小時」超大數字，脈衝動畫
- 三天用三個大色塊（Day 1 藍 / Day 2 綠 / Day 3 金）
- 「兩個都是贏」用雙綠色欄
- 日期承諾：手寫風格字體 + 空白底線
- 最後回到深色背景，收束感

## 講師備註

> 「為什麼是 72 小時？因為動力的半衰期大概 48 小時。如果你今天很興奮，但等到下週一才開始——你不會開始。」
> 「所以：今晚回家。打開 AI。跑 Phase 1。不管跑得好不好。先跑一次。」
> 「在心裡想一個日期。今天是 ___。72 小時後是 ___。那天之前，你要發出一封訊息。」

## 過場

→ 最後一張。安靜地切。
